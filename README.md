# Simple Calculator

## 项目简介

这是一个使用 Python 编写的命令行简单计算器项目。

这个项目的目标是练习 Python 基础语法，包括：

- 函数
- 条件判断
- 循环
- 用户输入
- 异常处理
- 模块拆分

## 功能说明

当前版本支持以下功能：

- 加法 `+`
- 减法 `-`
- 乘法 `*`
- 除法 `/`
- 输入错误时提示重新输入
- 支持连续计算

```text
simple_calculator/
